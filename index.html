About

After building my first React App, I became more and more interested in software development and decided to make a career change. I’ve enjoyed a 6-year career working for Pacific Gas and Electric Company, a large utility company. I love engineering and I was encouraged by my friends, who are working at high tech companies, to learn some basic programming courses to see if I’m interested in coding. I started with learning online coding classes, and then built my first React App, which is called Covid-19 tracker used to track the trends and cases daily.

I recently graduated from Springboard Coding Bootcamp which enriched my programming knowledge from front-end, back-end, and database. I learned a lot of useful and modern skills such as HTML, CSS, JavaScript, Node.js, Express, Python, SQL, React, Redux, and more. I enjoy learning new technologies to equip myself and working in a creative and passionate atmosphere.

With my coding abilities, along with my experience in the utility industry, I believe I’m ready to join a high-functioning and passionate software development team. My goal is to keep pushing myself to learn new things so that I may grow both as a developer and as an individual.

In my free time, I enjoying building Lego, photographing, and playing soccer (Of course including Video Game Pro Evolution Soccer).
